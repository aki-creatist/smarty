# データ型

## 概要

* データ型とは
    * データの種類
    * `変数のデータ型`はあらかじめ変数名の宣言時に一緒に指定する
        * 宣言された型の種類によりメモリ上に何区画分の領域を確保するのかが変わる
        * PHPでは、変数の型を宣言せずに使用可能
            * 型の設定はPHPが判断
                * 文字列を代入するとその変数は文字列型になる
                    * `文字列型`：`a`や`あ`など
                * 整数を代入すると整数型に自動的に変換される
                    * `整数型`：`1`や`3`など正の整数、０、負の整数
                    * `実数型`：`1.1`や`3.14`など
                        * `小数点以下の値を持つ数値`を表すデータ型
                        * `浮動小数点型`とも呼ばれる

浮動小数点型について

* `3.14`などは数の種類としては整数
* 小数点以下の表記があるので、`実数型`として扱われる

## 擬似言語で変数を宣言する

* 行頭に◯印</font>をつける
* 日本語で変数のデータ型を指定
* データ型の後に、コロン(:)と変数名を書く

```text
◯整数型:i
◯文字型:c
```

複数の変数を一行で宣言することも可能

```text
◯整数型:min, max
```
